Changelog

Every update to Slitheroo since launch. The game is actively maintained and improved based on player feedback and new ideas. Check back here to see what has changed in each version.

v2.0.0 — Latest

Released April 1, 2026

The Big Feature Update

Version 2.0 is the largest update to Slitheroo since launch, adding two new game modes, a full power-up system, food variety, achievements, a personal stats page, and major gameplay polish.

  • New Timed mode — score as much as possible in 60 seconds
  • New Daily Challenge mode — seeded board, same for everyone each day
  • New Power-ups: Shield, Score Multiplier (2×), Slow Motion, Ghost
  • New Three food types: Normal (10pts), Bonus (25pts), Mega (50pts)
  • New 12 achievements with localStorage persistence
  • New Personal stats page (stats.html) tracking games, scores, levels, food, power-ups
  • New Day streak tracker with on-screen display
  • New Screen shake on death
  • New Particle burst effects on food pickup
  • New Web Audio API sound effects — eat, level up, die, power-up, combo (no external files)
  • New Animated pulsing food with inner highlight
  • New Combo streak popups at x3 and above
  • New Share score as branded PNG image (canvas export)
  • New Challenge link — copies a URL showing your score for a rival to beat
  • New Sound toggle button in game header (M key shortcut)
  • New Power-up HUD bar displayed on canvas during active effects
  • New Timed mode countdown HUD on canvas
  • Improved Seeded RNG system for deterministic Daily Challenge boards
  • Improved Shield power-up absorbs wall and obstacle hits before applying normal death logic
  • Improved Ghost power-up bypasses self-collision checks
  • Improved Slow Motion multiplies the game loop accumulator delay rather than changing speed constants
  • Improved Service worker bumped to v2.0.0, stats.html added to cache
  • Improved Navigation updated across all pages to include Stats, Blog, FAQ links
v1.3.0

Released February 2026

Rewarded Ads & Continue Feature

  • New Rewarded ad integration via Google Publisher Tag (GPT)
  • New Continue-after-death feature — one continue per day via rewarded ad
  • New Consent banner for ad personalization (GDPR-aware)
  • New Non-personalized ad default with opt-in for personalized ads
  • Improved SLITHEROO_CONFIG object for centralized ad and backend configuration
  • Improved Grace steps on revival to prevent immediate re-death after continue
  • Improved Service worker updated to v1.4.1
v1.2.0

Released January 2026

Customization & Zen Mode

  • New Zen mode — walls wrap around instead of killing the snake
  • New Customization screen with four snake skins: Neon, Retro, Striped, Pulse
  • New Four visual themes: Deep Sea, Cyberpunk, Matrix, Sunset
  • New Zen mode toggle for showing obstacles
  • New All-time leaderboard (Top 50)
  • Improved Dynamic hue-rotate CSS variable for level-based visual theming
  • Improved Score bump animation on food pickup
  • Improved Leaderboard tab switching between daily and all-time views
v1.1.0

Released December 2025

Leaderboards & Nicknames

  • New Daily leaderboard via Supabase backend
  • New Nickname system with validation (3–16 chars, alphanumeric)
  • New Score submission after each ranked run
  • New Current player highlighting in leaderboard list
  • New Retry submit button on network failure
  • New Pregame stats showing best score and daily rank
  • Improved IP hash stored server-side for basic spam protection
  • Improved RLS policies on Supabase scores table
v1.0.0

Released November 2025

Initial Launch

  • New Core Snake gameplay on HTML5 Canvas
  • New Ranked mode with wall collision and obstacle scaling by level
  • New Level progression with speed increase every 50 points

  • New Mobile swipe controls with pointer event handling
  • New Splash screen and animated overlays

  • New High score persistence via localStorage
