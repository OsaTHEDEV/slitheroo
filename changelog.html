Changelog

This page records shipped work only. It is meant to show how the game and the support site have actually changed over time.

v2.0.1

Released April 25, 2026

Site Quality and Review Readiness

  • Rewrote homepage support copy to describe the real project instead of generic arcade language.
  • Added a dedicated leaderboard rules page.
  • Reworked the help page into practical troubleshooting content.
  • Replaced placeholder contact information with real publisher contact details.
  • Expanded privacy and terms pages to match actual product behavior.
  • Removed visible encoding issues from public-facing text.
  • Reduced non-essential ad-related loading on the homepage during the review push.
v2.0.0

Released April 1, 2026

Modes, Power-Ups, and Stats

  • Added Timed mode and Daily Challenge mode.
  • Added power-ups, multiple food types, and achievement tracking.
  • Added the personal stats page and day streak tracking.
  • Improved sharing with score export and challenge links.
  • Updated navigation so support pages were easier to reach from the game.
v1.3.0

Released February 2026

Continue Flow and Monetization Work

  • Added a continue flow tied to optional rewarded advertising.
  • Added ad-consent handling and non-personalized defaults where relevant.
  • Improved centralized configuration for backend and ad behavior.
v1.2.0

Released January 2026

Customization and Zen

  • Added Zen mode, skins, themes, and obstacle visibility options.
  • Added the all-time leaderboard view.
v1.1.0

Released December 2025

Live Scoreboards

  • Added nickname handling and score submission.
  • Added daily leaderboard support through Supabase.
  • Introduced basic anti-spam protections for competitive scoring.
