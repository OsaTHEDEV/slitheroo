Help and Troubleshooting

This page is for the practical issues players actually run into: controls not responding, missing local stats, leaderboard confusion, offline behavior, and mobile quirks.

The snake is not responding on mobile. What should I do?
Tap the game area once, then swipe clearly in one direction. Small accidental taps can be ignored by the gesture threshold. If the page was backgrounded and restored, reload once and try again.
My local stats or achievements are gone.
Stats, achievements, streaks, and local bests are stored in your browser. If you cleared site data, switched browsers, used private mode, or changed devices, those local records will not follow automatically.
Why did my score not appear on the leaderboard?
The most common reasons are network failure, nickname issues, or backend validation. Start by confirming you played a ranked-eligible run with a saved nickname. Then check the Leaderboard Rules page for submission basics.
Does Slitheroo work offline?
The game can cache core assets for repeat visits, which helps with quick loading and some offline use. Live leaderboard features still need a working connection because they rely on the backend.
How do I reset my progress?
Use the reset button on the My Stats page if you want to clear local stat tracking. If you need a full browser-level reset, remove site storage in your browser settings.
What should I include in a bug report?
Include the device, browser, what mode you were in, the last thing you did before the issue, and whether the problem happens every time. If it is a leaderboard issue, also include the nickname used and the score you expected.
Where can I read what changed recently?
Use the changelog for release history and the devlog for context and design notes.
